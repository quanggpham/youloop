Module Details — Part 1: Core Modules

Interfaces, responsibilities, and interactions

Module 1: YouTube Detector

Interface:

isWatchPage() → boolean
getVideoId() → string | null
onPageChange(callback) → cleanup
waitForPlayer() → Promise<HTMLVideoElement>

Responsibility:

  • Phát hiện SPA navigation của YouTube (YouTube không reload page khi chuyển video)
  • Extract video ID từ URL hoặc DOM
  • Đợi player element xuất hiện (YouTube load động)
  • Signal lifecycle: video-started, video-changed, player-unavailable

Edge Cases:

  • YouTube Shorts → không inject (khác cấu trúc DOM)
  • Embedded player (youtube.com/embed/*) → có hỗ trợ
  • Playlist autoplay → detect khi video ID thay đổi
Module 2: Loop Engine

Interface:

setLoop(startSec, endSec) → void
enable() / disable() → void
isActive() → boolean
getLoopPoints() → { start, end } | null
onLoop(callback) → cleanup  // fires on each A→B restart

Loop Algorithm:

// Core loop — runs via rAF when active
function tick() {
  const t = player.currentTime;
  if (t >= endSec || t < startSec) {
    player.currentTime = startSec;  // seamless seek
  }
  if (isEnabled) requestAnimationFrame(tick);
}

Edge Cases:

  • User manually seeks outside loop range → tạm thời cho phép, không force back
  • Video ended naturally → tự seek về start và play
  • Ad playing → tạm dừng loop, resume sau khi ad kết thúc
  • start > end (user nhập sai) → tự động hoán đổi hoặc báo lỗi
Module 3: Timeline UI

Interface:

inject() → void
destroy() → void
updateMarkers(start, end) → void
onSetStart(cb) / onSetEnd(cb) → cleanup
onToggleLoop(cb) → cleanup
onDragMarker(type, cb) → cleanup

UI Elements (injected into YouTube player):

[A↻B] Toggle Loop button — next to YouTube controls
[⏺ Set A] Set Start button — captures current video time
[⏹ Set B] Set End button — captures current video time
▎▎A ▁▁▁▁▁▁▁▁▁▁ B▎▎ Markers on progress bar
[1:30 — 3:45] Time display — shows current loop range

Visual Style:

  • Màu accent: #FF4081 (pink) — nổi bật trên nền đỏ YouTube
  • Marker trên timeline: 2 thanh dọc mỏng, kéo được
  • Loop region trên progress bar: highlight màu giữa 2 marker
  • Button: icon đơn giản, tooltip rõ ràng
